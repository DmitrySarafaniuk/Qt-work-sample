Цикл for

Цей Java цикл виглядає так: 

for (initialization; termination; increment) {
    statement(s)
}

Здесь:

  • initialization — вираз, який ініціалізує виконання циклу. Виконується тільки раз на початку циклу. Найчастіше в цьому виразі ініціалізується лічильник циклу
  • termination — boolean вираз, який регулює закінчення виконання циклу. Якщо результат выражения буде дорівнювати false, цикл for перерветься.
  • increment — вираз, який вмконується після кожної ітерації циклу. Найчастіше в цьому виразі відбувається інкрементування чи декрементування змінної лічильника.
  • statement(s) — тіло цикла.

Вирази initialization, termination, increment опційні. Якщо опустити кожен з них, ми отримаємо нескінченний цикл: 

// нескінченний цикл
for ( ; ; ) {
    // код тіла цикла
}

Приклад циклу for: 

public class ForExample {

    public static void main(String[] args) {
        for (int i = 1; i < 6; i++) {
            System.out.println("Рядок №" + i);
        }
    }
}

Вывод: 

Рядок №1
Рядок №2
Рядок №3
Рядок №4
Рядок №5
